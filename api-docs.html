Introduction

The UZMA Satellite Imagery API provides programmatic access to our archive of high-resolution Earth observation data. All requests go to:

Base URL:  https://api.uzma.com.my/v1
Format:    application/json
Dates:     ISO 8601  (YYYY-MM-DD)
Currency:  Malaysian Ringgit (MYR)

Authentication

Public endpoints (search, preview) require no authentication. Protected endpoints (order, download) require a JWT bearer token from POST /auth. Tokens expire after 24 hours.

# Attach to every protected request
Authorization: Bearer <your_jwt_token>

GET /images

Returns a paginated list of available satellite scenes.

GET/v1/images

Query Parameters

ParameterTypeDescription
search optstringLocation name or keyword
type optstringoptical, sar, multispectral, thermal
res_max optfloatMax resolution in meters
date_from optdateYYYY-MM-DD
date_to optdateYYYY-MM-DD
sort optstringnewest, price_asc, price_desc, res_asc
limit optintDefault 20, max 100
offset optintPagination offset

POST /auth

Authenticates with email and API password. Returns a JWT token.

POST/v1/auth

Code Snippets

cURL
Python
JavaScript
curl -X POST "https://api.uzma.com.my/v1/auth" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"your_api_key"}'
import requests

r = requests.post("https://api.uzma.com.my/v1/auth", json={
    "email": "user@example.com",
    "password": "your_api_key"
})
token = r.json()["token"]
headers = {"Authorization": f"Bearer {token}"}
const r = await fetch("https://api.uzma.com.my/v1/auth", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ email: "user@example.com", password: "key" })
});
const { token } = await r.json();

POST /order

Creates an order. Requires Bearer token. Returns signed download URLs.

POST/v1/order
# Request
{
  "items": [{ "id": "img_kl_20241115_opt" }],
  "payment_method": "quota"
}

# Response
{
  "order_id": "ord_20241210_abc123",
  "status": "completed",
  "total_myr": 850,
  "downloads": [{
    "download_url": "https://cdn.uzma.com.my/dl/signed_xyz",
    "expires_at": "2024-12-11T10:00:00Z"
  }]
}

GET /download/:token

Downloads full-resolution GeoTIFF. Token is time-limited (24h) and single-use.

GET/v1/download/:token
Content-Type:        image/tiff
Content-Disposition: attachment; filename="kl_20241115_0.5m.tif"
