Introduction

The UZMA Satellite Imagery API provides programmatic access to our archive of high-resolution Earth observation data covering Malaysia and surrounding regions. All responses are JSON; dates use ISO 8601; currency is Malaysian Ringgit (MYR).

Base URL:  https://api.uzma.com.my/v1
Format:    application/json
Dates:     ISO 8601  (YYYY-MM-DD)
Currency:  Malaysian Ringgit (MYR)
Quick Start: Public endpoints (search, preview) require no authentication. For ordering and downloads, obtain a JWT token via POST /auth.

Authentication

Protected endpoints require a JWT Bearer token from POST /auth. Tokens expire after 24 hours.

# Include in all protected request headers
Authorization: Bearer <your_jwt_token>

Token Lifecycle

Tokens are single-session and expire at 24 h. Store securely — never expose in client-side code or public repositories. Rotate access keys from your dashboard at uzma.com.my.

GET /images

Returns a paginated list of available satellite scenes. All parameters are optional.

GET/v1/images

Query Parameters

ParameterTypeDescription
search optstringLocation name or keyword
type optstringoptical · sar · multispectral · thermal
res_max optfloatMax resolution in metres (e.g. 1.0)
date_from optdateYYYY-MM-DD
date_to optdateYYYY-MM-DD
price_max optintMaximum price in MYR
sort optstringnewest · price_asc · price_desc · res_asc
limit optintResults per page. Default: 20, max: 100
offset optintPagination offset

      "bands": "RGB",
      "date": "2024-11-15",
      "price_myr": 850,
      "preview_url": "https://cdn.uzma.com.my/previews/..."
    }
  ]
}

POST /auth

Authenticates and returns a JWT token valid for 24 hours.

POST/v1/auth

Request Body

FieldTypeDescription
email reqstringYour UZMA account email
password reqstringYour API access key
cURL
Python
JavaScript
curl -X POST "https://api.uzma.com.my/v1/auth" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"your_api_key"}'
import requests

r = requests.post("https://api.uzma.com.my/v1/auth", json={
    "email": "user@example.com",
    "password": "your_api_key"
})
token = r.json()["token"]
headers = {"Authorization": f"Bearer {token}"}
const r = await fetch("https://api.uzma.com.my/v1/auth", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ email: "user@example.com", password: "key" })
});
const { token } = await r.json();

POST /order

Creates a purchase order. Requires Bearer token. Returns signed download URLs valid for 24 hours.

POST/v1/order

Request Body

FieldTypeDescription
items reqarrayArray of {id} objects
payment_method optstringquota (default) · invoice
# Request body
{
  "items": [{ "id": "img_kl_20241115_opt" }],
  "payment_method": "quota"
}
200application/json
{
  "order_id": "ord_20241210_abc123",
  "status": "completed",
  "total_myr": 850,
  "downloads": [{
    "image_id": "img_kl_20241115_opt",
    "download_url": "https://cdn.uzma.com.my/dl/signed_xyz",
    "expires_at": "2024-12-11T10:00:00Z"
  }]
}

GET /download/:token

Downloads the full-resolution GeoTIFF. Tokens are single-use and expire after 24 hours.

GET/v1/download/:token
Content-Type:        image/tiff
Content-Disposition: attachment; filename="kl_20241115_0.5m.tif"
Content-Length:      248123456
Note: Download tokens are single-use and expire after 24 hours. Monitor the expires_at field from your order response and re-order to obtain a new token.

Error Codes

All errors return a JSON body with error and message fields.

HTTPStatusDescription

401UnauthorizedMissing or expired JWT token

Webhooks
